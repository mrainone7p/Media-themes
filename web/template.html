Set up connections, choose library scope, control source discovery, and tune downloads.
Media Tracks works best when this page is set up in order: connect Plex, choose the movie or TV libraries you want to manage, choose how it should find sources, then tune output and performance.

Pipeline Steps
Toggle which parts of the automated pipeline should run. Manual "Run Now" buttons work regardless of these settings.
Step 1 Scan Libraries
Connects to Plex, indexes all media, checks which folders already have a theme file. Media without a theme is set to PENDING. Media with a theme is set to AVAILABLE. Uses: Plex. No YouTube or TMDB calls. Fast.
Step 2 Find Theme Sources
Searches sources for a theme based on the configured search logic for each PENDING media item. Saves the URL and moves status to STAGED. Uses: TMDB + YouTube or Golden Source if configured.
Step 3 Download Themes
Downloads audio for every APPROVED media item, saves the theme file into the media folder, and moves status to AVAILABLE. Uses: yt-dlp + ffmpeg.
Auto-Approve
Automatically approve all STAGED media before downloading — fully hands-off pipeline
