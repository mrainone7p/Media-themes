SOURCE DISCOVERY
˅
These settings control how sources are found for each MISSING theme.
Search setup
Choose how source discovery searches for and validates theme sources.
Playlist mode looks for soundtrack playlists. Direct mode looks for a single theme or song result.
Strict matching checks that the source includes the full query. Fuzzy matching uses the query and accepts the first result.
If the primary search mode finds no valid strict match, try the other mode once.
Manual search selections skip STAGED and go straight to APPROVED.
Advanced query templates
Template for playlist mode. Use {title} and {year}.
Template for direct mode. Use {title} and {year}.
CURATED SOURCE DISCOVERY
˅
A curated list of trusted theme matches. It can speed up matching and improve result quality for supported titles, though some gaps may remain depending on your library.
Golden Source
Point to a trusted source CSV, then test it before using it for source matching.
Required: tmdb_id, source_url. Optional: title, year, start_offset, updated_at, notes.
Golden Source mode
Use only the curated list when you want faster, more controlled matching.
Disable YouTube source discovery and only match against the curated source list. This can be faster, but titles not covered by your list will remain unmatched.
Refresh the Golden Source list before each source-matching run.

STATUS KEY · Step 1
⟳ Scan Libraries
Connects to Plex, indexes media, and checks whether each folder already has a local theme file. Items without a theme are marked Missing. Items with a theme are marked Available. Uses Plex only. Fast.
AUTO SCHEDULE · Step 2
⚡ Find Theme Sources
Searches for theme sources for each Missing item using the configured search logic. When a source is found, the item moves to Staged. Items marked Unmonitored or Failed are ignored. Uses TMDB, YouTube, or Golden Source when configured.
PIPELINE STEPS · Step 3
↓ Download Themes
Downloads themes for each Approved item, saves the theme file into the media folder, and moves the item to Available. Items marked Unmonitored or Failed are ignored. Uses yt-dlp and ffmpeg.

Connects to Plex, indexes media, and checks whether each folder already has a local theme file. Items without a theme are marked Missing. Items with a theme are marked Available. Uses Plex only. Fast.
Step 2 Find Theme Sources
Searches for theme sources for each Missing item using the configured search logic. When a source is found, the item moves to Staged. Items marked Unmonitored or Failed are ignored. Uses TMDB, YouTube, or Golden Source when configured.
